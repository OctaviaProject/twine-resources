You wake up literally inside the Internet, in a Twine tutorial. You brush some ones and zeros off of your face. What do you do?

$ learn twine

OMG perfect choice.

Setup

This tutorial is written for Twine 1.4.2. Get started by downloading Twine (Windows, Mac). The screenshots will be of the Mac version of Twine, but everything translates directly to the Windows version.

$ wth is twine

If you're looking for background on Twine, see the official site, where you can download the software and find lots of other resources. In a nutshell, Twine is software for creating interactive fiction, games, poetry, visual novels, text adventures, and entire galaxies.

Note: most of this tutorial will not work in Twine 2. If you want something fun to do, rewrite the whole thing for Twine 2!

What to expect

We're just gonna learn how to make Twine stories bit by bit, and hopefully get into the more interesting aspects of Twine, such as custom formats, CSS, and JavaScript. There will be explanations, examples, and a condensed cheat sheet. This tutorial is suitable for people without any experience with Twine, CSS, or JavaScript.
